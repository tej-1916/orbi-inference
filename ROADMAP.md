# ORBI Roadmap

## v0.1 — Control plane foundation

Secure gateway, durable queue, worker identity, budgets, idempotency, leases, and tests.

## v0.2 — Local ORBI Node

Local/Docker model worker, llama.cpp adapter, model loading, generation, and worker heartbeats.

## v0.3 — Notebook workers

Experimental Colab worker and asynchronous Kaggle worker with explicit provider-limit warnings.

## v0.4 — Managed fallback

Budget-gated Hugging Face adapter, routing policy, provider scoring, and failover incidents.

## v0.5 — Streaming and SDK

SSE streaming, `/v1/jobs`, Python SDK, and OpenAI client examples.

## v0.6–v0.9 — Operations and ecosystem

Observability, n8n reports, console, TypeScript SDK, integrations, and production hardening.

## v1.0 — Stable inference fabric

Stable worker protocol, production deployment references, security review, and compatibility policy.
