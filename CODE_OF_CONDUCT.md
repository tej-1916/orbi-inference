# Code of Conduct

ORBI contributors must communicate respectfully, review technical ideas on their merits, protect
user privacy, and never pressure others to disclose credentials or private infrastructure details.
Harassment, discrimination, threats, deliberate security abuse, and publication of private
vulnerability details before coordinated disclosure are not accepted.

Report conduct concerns privately to the maintainers. Maintainers may remove content, restrict
participation, or ban contributors when necessary to protect the community and project.
